## 📖 项目简介

欢迎来到我的 **ROS 2 (Robot Operating System 2)** 学习笔记仓库。

作为一个 ROS 2 的初学者，我在学习过程中发现，虽然官方文档非常详尽，但中文资料相对分散，且部分概念对于新手来说稍显晦涩。为了巩固所学知识，同时也为了方便日后查阅，我决定启动这个文档项目。

本项目基于 [MkDocs](https://www.mkdocs.org/) 和 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 构建，旨在提供一个**结构清晰、阅读体验良好**的中文知识库。

## 📚 内容大纲

目前的文档主要覆盖（或计划覆盖）以下内容：

* **基础篇 (Basics):** 工作空间、节点 (Node)、话题 (Topic)、服务 (Service)、动作 (Action) 等核心概念。
* **进阶篇 (Intermediate):** Launch 文件编写、TF2 坐标变换、参数服务器。
* **开发实战 (Dev):** C++ 与 Python 的混合编程、自定义接口 (msg/srv/action)。
* **常用工具 (Tools):** Rviz2, Gazebo, Rosbag2 的使用技巧。
* **算法集成 (Integration):** Nav2 (导航) 与 MoveIt2 (机械臂) 的初步探索。
