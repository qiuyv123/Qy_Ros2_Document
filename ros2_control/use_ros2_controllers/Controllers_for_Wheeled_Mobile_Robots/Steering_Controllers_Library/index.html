    Steering Controllers Library（转向控制器库）¶

    steering_controllers_library 是一个面向带转向驱动的移动机器人（2 自由度）的控制器库，适用于具有非完整约束（non-holonomic constraints）的系统。

    • 实现了通用的里程计计算与更新逻辑
    • 定义了标准接口
    • 仅使用逆运动学（inverse kinematics），不包含反馈控制回路（如路径跟踪等高级控制）

    📚 有关移动机器人运动学基础及本文使用的术语，请参阅：轮式移动机器人运动学（Wheeled Mobile Robot Kinematics）

    控制器执行逻辑¶

    • 输入：速度指令（Twist 或 TwistStamped 消息）
    • 仅使用 线速度 x 分量 和 角速度 z 分量
    • 其他分量被忽略
    • 链式模式（chained mode）下，提供两个参考接口：
    • 线速度参考
    • 转向角位置参考
    • 其他特性：
    • 支持前轮转向和后轮转向配置
    • 发布 Odometry 和 TF 消息
    • 基于参数的输入命令超时机制
    • 轮子命令通过里程计库计算，根据具体运动学模型生成牵引（traction）和转向（steering）指令

    当前支持的运动学模型¶

    模型 转向关节 驱动关节
    Bicycle（自行车模型） 1 个 1 个
    Tricycle（三轮模型） 1 个 2 个
    Ackermann（阿克曼模型） 2 个 2 个

    控制器接口说明¶

    参考接口（References，来自前置控制器）¶

    仅在链式模式下使用（in_chained_mode == true）：

    接口 类型 单位 说明
    <controller_name>/linear/velocity double m/s 本体线速度
    <controller_name>/angular/velocity double rad/s 本体角速度

    表示机器人本体速度（body twist）。

    命令接口（Command Interfaces）¶

    根据 front_steering 参数决定：

    若 front_steering == true（前轮转向）¶

    • <front_wheels_names[i]>/position（rad）：前轮转向角
    • <rear_wheels_names[i]>/velocity（rad/s）：后轮驱动速度

    若 front_steering == false（后轮转向）¶

    • <front_wheels_names[i]>/velocity（rad/s）：前轮驱动速度
    • <rear_wheels_names[i]>/position（rad）：后轮转向角

    状态接口（State Interfaces）¶

    反馈类型由 position_feedback 决定： - position_feedback == true → 牵引轮使用 位置反馈（position） - position_feedback == false → 牵引轮使用 速度反馈（velocity）

    若 front_steering == true¶

    • 前轮：<front_wheels_names[i]>/position（rad）
    • 后轮：<rear_wheels_names[i]>/<TRACTION_FEEDBACK_TYPE>（rad 或 rad/s）

    若 front_steering == false¶

    • 前轮：<front_wheels_names[i]>/<TRACTION_FEEDBACK_TYPE>（rad 或 rad/s）
    • 后轮：<rear_wheels_names[i]>/position（rad）

    注：TRACTION_FEEDBACK_TYPE 由 position_feedback 决定。

    ROS 2 接口¶

    订阅者（Subscribers）¶

    仅在非链式模式下使用（in_chained_mode == false）：

    • <controller_name>/reference（geometry_msgs/msg/TwistStamped）
      当 use_stamped_vel == true 时使用。
    • <controller_name>/reference_unstamped（geometry_msgs/msg/Twist）
      当 use_stamped_vel == false 时使用。

    发布者（Publishers）¶

    • <controller_name>/odometry（nav_msgs/msg/Odometry）
      机器人位姿与速度估计。
    • <controller_name>/tf_odometry（tf2_msgs/msg/TFMessage）
      里程计 TF 变换。
    • <controller_name>/controller_state（control_msgs/msg/SteeringControllerStatus）
      控制器状态（如当前转向角、速度等）。

    参数（Parameters）¶

    该库使用 generate_parameter_library 管理参数。完整示例见控制器包的 test/ 目录。

    核心参数¶

    参数 类型 默认值 说明
    reference_timeout double 1.0 参考命令超时时间（秒），超时后重置。设为 0 表示每次运行后重置。
    front_steering bool true 是否为前轮转向？（只读）

    轮子关节名称¶

    参数 默认值 约束 说明
    rear_wheels_names — 非空、无重复、长度 < 5（只读） 后轮关节名
    front_wheels_names — 非空、无重复、长度 < 5（只读） 前轮关节名
    rear_wheels_state_names {} 长度 < 5，无重复（只读） 后轮状态读取关节名（可选）
    front_wheels_state_names {} 长度 < 5，无重复（只读） 前轮状态读取关节名（可选）

    若状态关节名未指定，则使用命令关节名。

    控制与里程计算¶

    参数 默认值 说明
    open_loop false 里程计是否使用开环计算（基于命令而非反馈）
    reduce_wheel_speed_until_steering_reached false 在转向角未到位前降低轮速
    velocity_rolling_window_size 10 计算里程计速度时的滑动窗口大小
    position_feedback false 是否使用位置反馈（否则使用速度反馈）
    use_stamped_vel false 是否使用带时间戳的速度消息

    坐标系与协方差¶

    参数 默认值 说明
    base_frame_id "base_link" 机器人基座坐标系
    odom_frame_id "odom" 里程计坐标系
    enable_odom_tf true 是否发布 odom → base_link 的 TF
    twist_covariance_diagonal {0.0, 7.0, 14.0, 21.0, 28.0, 35.0} 速度协方差对角线（6维）
    pose_covariance_diagonal {0.0, 7.0, 14.0, 21.0, 28.0, 35.0} 位姿协方差对角线（6维）
